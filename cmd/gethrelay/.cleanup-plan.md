# Cleanup Plan for gethrelay Codebase

## Dependencies Analysis

gethrelay needs:
- Core: `common`, `core`, `forkid`, `rawdb`, `triedb`, `params`
- P2P: `p2p`, `p2p/enode`, `p2p/nat`
- Node: `node`
- RPC: `rpc`
- Types: `types`
- Logging: `log`
- Internal: `internal/debug`, `internal/flags`
- CLI: `urfave/cli/v2`
- Relay: `eth/relay`, `eth/protocols/eth` (relay backend)

## Safe to Remove

1. Remove command binaries first
2. Test build
3. Remove unused packages only if they break nothing
